# wonkle.io

## Setup

1. [Setup devenv](https://devenv.sh/getting-started)
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Create `.env.local` from `.env.local.example`
4. Setup Convex (only required once):
   ```bash
   pnpx convex dev
   ```
5. Start dev server:
   ```bash
   pnpm dev
   ```
